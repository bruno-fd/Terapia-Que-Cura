## Contexto e objetivo

Esta task implementa uma arquitetura de dois níveis de categorização na plataforma. O usuário interage com subcategorias via busca livre (digitação ou seleção). O advogado marca macrocategorias obrigatoriamente e subcategorias de forma opcional e expansível. A busca retorna resultados em camadas de relevância.

### 1. main.js

Substituir `CATEGORIAS` por `CATEGORIAS_COMPLETAS` conforme acima.

Criar função auxiliar que retorna apenas os nomes das macrocategorias para uso em selects e filtros simples:

```javascript
function getMacrocategorias() {
  return CATEGORIAS_COMPLETAS.map(c => ({ nome: c.nome, slug: c.slug, emoji: c.emoji }));
}
```

Criar função de busca que pesquisa em macrocategorias e subcategorias simultaneamente:

```javascript
function buscarCategorias(termo) {
  const t = normalizar(termo);
  const resultados = [];

  CATEGORIAS_COMPLETAS.forEach(cat => {
    // Verifica se o termo bate com a macrocategoria
    if (normalizar(cat.nome).includes(t)) {
      resultados.push({ tipo: 'macro', nome: cat.nome, slug: cat.slug, emoji: cat.emoji });
    }
    // Verifica subcategorias
    cat.subcategorias.forEach(sub => {
      if (normalizar(sub).includes(t)) {
        resultados.push({
          tipo: 'sub',
          nome: sub,
          macro_nome: cat.nome,
          macro_slug: cat.slug,
          emoji: cat.emoji
        });
      }
    });
  });

  return resultados;
}

function normalizar(str) {
  return str.normalize('NFD').replace(/[\u0300-\u036f]/g, '').toLowerCase();
}
```

### 5. painel-perfil.html — Seção de áreas de atuação

Esta é a mudança mais importante para o advogado. Substituir a seção atual de badges simples por um sistema expansível em dois níveis.

**Comportamento:**

Exibir os 12 badges de macrocategoria como antes. Ao clicar em uma macrocategoria para ativá-la, ela se expande abaixo e revela as subcategorias daquela categoria.

Layout da expansão:
- Badge da macrocategoria fica ativo (azul)
- Abaixo dele, aparece com animação suave um bloco expansível com fundo `--color-primary-50`, `--radius-md`, padding `--space-4`
- Título do bloco: "Quer detalhar suas especialidades em [nome da macrocategoria]?" — font-size-sm, `--color-neutral-600`
- Texto secundário: "Opcional. Ajuda clientes a te encontrarem com mais precisão." — font-size-xs, `--color-neutral-500`
- Subcategorias como badges menores, mesmo sistema de toggle (ativo/inativo)

Ao desmarcar a macrocategoria, o bloco de subcategorias colapsa e todas as subcategorias daquela categoria são desmarcadas automaticamente.

É possível ter a macrocategoria ativa sem nenhuma subcategoria marcada — isso é o comportamento padrão e esperado.

**Estado visual dos badges de subcategoria:**
- Ativo: fundo `--color-primary-100`, texto `--color-primary-800`, border `--color-primary-400`
- Inativo: fundo branco, texto `--color-neutral-500`, border `--color-neutral-300`
- Tamanho menor que os badges de macrocategoria: padding 4px 10px, font-size-xs

**Estado inicial de demonstração:**
