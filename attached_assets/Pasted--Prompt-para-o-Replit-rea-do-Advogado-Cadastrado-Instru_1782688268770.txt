# Prompt para o Replit — Área do Advogado Cadastrado

---

## Instrução geral

Crie as seguintes páginas para a área logada do advogado no site Minha Causa Justa:

- `login.html` — página de login
- `painel-perfil.html` — edição do perfil do advogado
- `painel-metricas.html` — desempenho do perfil
- `painel-assinatura.html` — gestão da assinatura

Use o mesmo `style.css` e `main.js` já existentes. Adicione apenas os estilos necessários para o painel diretamente no `style.css`, sem sobrescrever nada do design system existente. Toda a lógica de estado (login simulado, salvar dados, status da assinatura) deve ser feita em JavaScript puro, sem bibliotecas externas.

## Instruções técnicas gerais

- Todo o estado (usuário logado, dados do perfil, status da assinatura, métricas) deve ser simulado em JavaScript puro com dados fictícios hardcoded. Sem backend real nesta etapa.
- Na página de métricas, simular dois estados via toggle de demonstração no topo: "Primeiro mês" e "3 meses de cadastro". Isso permite visualizar os dois estados da interface sem backend.
- O toggle de demonstração de status (Ativa | Atrasada | Inativa) na página de assinatura serve apenas para visualizar as variações — em produção será controlado pelo webhook do Asaas.
- O seletor de período (Este mês | Últimos 3 meses | Total) na página de métricas deve trocar os números fictícios via JavaScript puro.
- Proteção de rota simulada: se acessar qualquer página do painel sem estar "logado", redirecionar para `login.html`. Simular com localStorage: ao fazer login, gravar `logado=true`; ao sair, limpar e redirecionar.
- Responsivo obrigatório. Menu lateral vira overlay em mobile (< 768px).
- Todos os modais fecham ao clicar fora ou no botão X.
- Scroll suave e foco no primeiro campo com erro ao tentar salvar com campos obrigatórios vazios.
- O gráfico de barras da página de métricas deve ser feito em SVG puro ou Canvas — sem nenhuma biblioteca externa.
- Código limpo, organizado e comentado por seção.
- Não usar nenhuma biblioteca externa além da fonte Inter do Google Fonts já importada.
